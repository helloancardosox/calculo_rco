# Calculadora RCO INSS

Projeto em HTML puro, pronto para publicar no GitHub Pages.

## Como publicar

1. Crie um repositorio no GitHub.
2. Envie os arquivos `index.html` e `README.md`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Salve e aguarde alguns minutos.

O link publico vai ficar neste formato:

`https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/`

## Arquivos

- `index.html`: aplicacao principal
- `README.md`: instrucoes de publicacao
